Projeto MarkGol - Desenvolvimento Web: Criação de sites responsivos

O projeto MarkGol é uma plataforma inovadora que facilita o agendamento de horários para jogar futebol em diversos campos. No nosso site, os usuários têm acesso a uma variedade de opções de locais, horários disponíveis e tipos de campo, permitindo que escolham o que melhor se adequa às suas necessidades e preferências. Com uma interface intuitiva e fácil de usar, MarkGol oferece um processo de reserva rápido e seguro, garantindo que você possa marcar seu jogo de forma eficiente e sem complicações. Seja para uma partida casual com amigos ou para jogos mais competitivos.

Projeto DrinkDossiê - Projeto Bimestral (PI I)

DrinkDossiê é um aplicativo desenvolvido para amantes de bebidas alcoólicas que desejam conhecer mais sobre suas opções antes de realizar uma compra. No DrinkDossiê, os usuários podem acessar avaliações detalhadas de diversas bebidas, desde vinhos e cervejas artesanais até destilados e coquetéis prontos. A plataforma oferece opiniões de outros consumidores, incluindo classificações de sabor, qualidade e valor.
